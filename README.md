# 백수생존기 (NEET Tycoon) — v2
- 상단 바에 **게임 방법** 버튼 추가 (모달로 규칙/팁 표시)
- 어두운 배경에서 버튼/텍스트가 검게 보이던 현상 개선
  - `.btn:disabled` 색상/불투명도/커서 지정
  - 모달 전반 텍스트 색상 고정

## 파일
- `index.html` : 마크업과 UI 구조 (+ 게임 방법 버튼)
- `styles.css` : 다크 네온 테마, 대비 개선
- `script.js`  : 게임 로직 + 도움말 모달

## 실행
같은 폴더에 두고 `index.html`을 브라우저로 엽니다. 저장은 로컬스토리지 `baeksu-v1` 키를 사용합니다.
